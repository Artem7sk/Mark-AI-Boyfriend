# 🕶️ Mark AI — The Ultimate Boyfriend Bot

**Mark AI** — это продвинутый Telegram-бот, имитирующий виртуального парня с использованием OpenAI. Проект включает в себя полноценную систему монетизации через **Telegram Stars**, VIP-статусы и админ-панель для управления пользователями.

---

## 🔥 Ключевые особенности / Features

* **🧠 AI Powered:** Интеграция с OpenAI для живого и контекстного общения.
* **⭐ Telegram Stars:** Поддержка новой системы платежей Telegram для покупки VIP и контента.
* **👑 VIP System:** Ограничение функционала для обычных пользователей и эксклюзивные возможности для VIP.
* **🖼️ Photo Puzzles:** Механика открытия "горячего" контента по частям после оплаты.
* **📊 Admin CRM:** Уведомления о регистрациях и покупках в реальном времени с прямыми ссылками на профили.
* **📂 Database:** Асинхронная работа с SQLite (aiosqlite).

## 🛠 Установка / Installation

1. Клонируйте репозиторий:
   ```bash
   git clone [https://github.com/ТВОЙ_ЛОГИН/Mark-AI-Boyfriend.git](https://github.com/ТВОЙ_ЛОГИН/Mark-AI-Boyfriend.git)
Установите зависимости:

Bash

pip install -r requirements.txt
Создайте файл .env и добавьте свои ключи:

Фрагмент кода

BOT_TOKEN=твой_токен_бота
OPENAI_API_KEY=твой_ключ_openai
ADMIN_ID=твой_id
Запустите бота:

Bash

python main.py
